# IADT_IAGEN

Repositório com materiais práticos da disciplina de **IA Generativa**, organizado por aulas e focado em fundamentos, arquiteturas neurais e aplicações com LLMs.

## Objetivo

Este projeto reúne notebooks para estudo e experimentação de técnicas de IA generativa, incluindo:

- Fundamentos de IA generativa
- Embeddings
- Redes neurais (MLP, CNN, RNN)
- GANs (Redes Adversárias Generativas)
- Introdução a Transformers e RAG

## Como usar

1. Abra a pasta do projeto no VS Code.
2. Instale as extensões recomendadas:
   - Python
   - Jupyter
3. Se necessário, crie e selecione um ambiente virtual Python.
4. Abra os arquivos `.ipynb` e execute as células em ordem.

## Requisitos sugeridos

- Python 3.10+
- Jupyter Notebook / JupyterLab
- Bibliotecas comuns de IA/ML (instaladas conforme necessidade de cada notebook)

## Observações

- Alguns notebooks podem exigir download de datasets ou instalação de pacotes adicionais.
- O arquivo `wonderland.txt` é utilizado em atividades com redes neurais recorrentes.
